Here are a few theorems to examine to gain
a better understanding of how logical opp-
erators work in Discrete Mathematics.

1. Double Negative law: Similar to Java
a !(!statement) will return true, because
you are negating the statement, so would
~(~int) in DS.

2.Commutative Laws: Similar to Java when
(thisStament && otherStatement) is the same
as (otherStatement && thisStatement), in
DS, similarly, a^b is logically equivalent
to b^a.

3.Idempotent laws: In Java, given a return
of statement string; string && string will
return string. Similarly in DS, statment
q^q will logically equal q.
